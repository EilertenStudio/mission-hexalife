## How contribute

1. Navigate in a web browser at the [online editor](https://encrier.ink/);
2. Copy and paste in the web browser the content of the [ink file](./mission-hexalife-v1.ink);
3. EDIT the story
4. Copy and paste the content of the web browser in the [ink file](./mission-hexalife-v1.ink) or create a new version;
5. Commit and push!
